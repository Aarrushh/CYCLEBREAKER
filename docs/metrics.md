# Metrics

Acquisition
- Onboarding completion rate
- Profile completeness

Engagement
- CTR on recommended items
- Save/Hide ratios
- Time-to-first-meaningful-opportunity

Quality
- Precision@K (human-labeled relevance)
- Coverage (per category/region)
- Freshness (avg days since last checked)

Outcomes
- Application starts/completions (if tracked)
- Self-reported benefit: “Was this useful?”

System
- Crawl success rate and latency
- Extraction accuracy vs ground truth
- Eligibility rule validation accuracy
